# Bus Reservation System

A simple Java console-based application for bus ticket booking and cancellation.

## Features
- Book bus tickets
- Cancel booked tickets
- Basic bus and passenger handling

## Technologies Used
- Java
- OOP concepts

## How to Run
1. Clone the repository:
   git clone https://github.com/bharathr-dev/Bus-Reservation-System.git 

2. Compile the program:
   javac *.java

3. Run the application:
   java BusMain
